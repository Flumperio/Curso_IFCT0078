Elementos obsoletos o en desuso en HTML

¡Atención! Las etiquetas y atributos que verás abajo no se deben usar en proyectos modernos. Muchos navegadores las soportan solo para compatibilidad y no forman parte de HTML5.

Ejemplos de etiquetas obsoletas

  • <font> — para cambiar color/tamaño de texto
    Esto es texto con <font> (no recomendado)
    En su lugar usa CSS: style="color:red; font-size:2em;"
  • <center> — para centrar contenido
    Esto está centrado con <center> (no recomendado)

    En su lugar usa CSS: style="text-align:center;"
  • <marquee> — texto en movimiento
    ¡Texto que se mueve! (no recomendado)
    Usa animaciones CSS o JavaScript moderno.
  • <big>, <small> (solo <small> es válido en HTML5, pero <big> ya no)
    Texto grande con <big> (no recomendado)
    Usa CSS: style="font-size:large;"
  • <strike> — texto tachado
    Texto tachado (no recomendado)
    Usa <del> o CSS: style="text-decoration: line-through;"
  • <u> — subrayado como presentación (en HTML5 es semántico solo en textos no latinos)
    Texto subrayado
    En la mayoría de los casos, usa CSS: style="text-decoration: underline;"
  • <applet>, <acronym>, <blink>, <bgsound>, <dir>, <frame>, <frameset>, <noframes>, <tt>, etc.
    No usar nunca. Busca alternativas semánticas y modernas.

Atributos obsoletos

  • align — para alinear texto, imágenes o tablas.
    Usa CSS: style="float:right;" o style="text-align:right;"
  • bgcolor — para color de fondo en tablas o celdas.
    Celda
    Usa CSS: style="background:#e3f2fd;"
  • border en tablas.
    Con borde
    Usa CSS: style="border: 3px solid #1976d2;"
  • width y height como atributos en <table>, <td>, etc.
    Usa CSS para controlar dimensiones.

¿Por qué no debes usar elementos obsoletos?

  • No son accesibles ni semánticos.
  • Pueden dejar de funcionar en navegadores futuros.
  • No son compatibles con buenas prácticas de CSS y HTML5.
  • Dificultan el mantenimiento y el posicionamiento (SEO).

Mini-reto

  1. Transforma una línea con <font color="blue"> en código moderno usando <span style="color:blue;">.
  2. Convierte una tabla con border y bgcolor a CSS.
  3. Busca una alternativa a <center> y prueba a centrar un título con CSS.
