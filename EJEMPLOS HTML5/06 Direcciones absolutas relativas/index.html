Direcciones absolutas y relativas en HTML

Las rutas se usan para enlazar o cargar recursos como imágenes, páginas y archivos. Hay varios tipos:

1. Dirección absoluta

src="https://www.w3schools.com/html/img_chania.jpg"

Enlace absoluto a Wikipedia

Siempre incluye protocolo y dominio (funciona desde cualquier web).

2. Dirección relativa (misma carpeta)

src="foto-local.jpg"

Enlace relativo a otra página del mismo directorio

La ruta se interpreta desde donde está guardado el archivo HTML actual.

3. Dirección relativa (subcarpeta)

src="imagenes/paisaje.jpg"

Busca el recurso dentro de la subcarpeta imagenes/ respecto a este archivo.

4. Dirección relativa (carpeta anterior)

src="../logo.png"

Sube una carpeta respecto a la ubicación actual antes de buscar el archivo.

5. Dirección relativa desde la raíz del sitio

src="/assets/img/logo.png"

Empieza desde la raíz del servidor web (no desde la raíz de tu disco).

6. Enlaces combinados

Página relativa | Página relativa desde la raíz | Página absoluta

7. Uso de <base> para cambiar la base de rutas relativas

<base href="https://www.ejemplo.com/blog/">
(Si se incluye esta etiqueta en el <head>, todas las rutas relativas se buscarán a partir de esa URL base.)

Mini-reto

  1. Agrega una imagen usando una ruta relativa de una subcarpeta "imagenes/".
  2. Agrega un enlace a una página que esté un nivel por encima de la actual (usa "../").
  3. Agrega un enlace absoluto a un recurso externo de tu elección.
